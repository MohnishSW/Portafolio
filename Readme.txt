# 🌐 Mohnish Portfolio

A modern personal portfolio website built using **HTML, CSS, and JavaScript**, designed to present my profile as a Junior Data Analyst with a clean dark-themed UI and animated UI components.

---

## 🎨 Design Overview

This portfolio uses a **dark UI theme** with green accent styling and smooth visual effects.

Key design elements:
- Dark background system using CSS variables
- Green accent color (`#2ecc71`)
- Glassmorphism-style cards with blur effects
- Smooth hover transitions and animations
- Floating profile image animation
- Gradient text effects in hero section

---

## 🧠 Sections

### 🏠 Hero Section
- Name: **Mohnish Hemdev**
- Role: Junior Data Analyst
- Animated circular profile image
- Floating animation effect (`@keyframes float`)
- Gradient heading text
- CTA button linking to LinkedIn

---

### 🛠️ Skills Section
A responsive grid showcasing technical skills:

- Data Analysis → Python, SQL, Excel  
- Data Visualization → Power BI, Dashboards, Reporting  
- Data Handling → SQL Queries, Data Cleaning, Transformation  

Each skill card includes:
- Font Awesome icon
- Hover lift animation
- Accent glow border effect

---

### 🔗 Navigation Bar
Fixed top navigation with:
- Smooth hover underline animation
- Links to:
  - Home (Hero section)
  - Skills section
  - Projects (Google Drive folder)
  - Contact (LinkedIn)

Includes:
- Glass blur effect (`backdrop-filter`)
- Sticky positioning
- Responsive mobile hiding (nav hidden under 768px)

---

### 📣 Footer
Includes:
- Copyright
- Social media links with hover animations:
  - GitHub
  - LinkedIn
  - Google Drive (Projects)
  - HackerRank

---

## 🛠️ Tech Stack

- HTML5 (semantic layout)
- CSS3 (modern styling + animations)
- JavaScript (external file: `script.js`)
- Font Awesome (icons via CDN)

---

## ⚙️ Styling Features (CSS Highlights)

- CSS Variables (`:root`)
  - Dark theme system
  - Accent color management
- Glassmorphism effects
- Responsive grid layout (`auto-fit, minmax`)
- Smooth transitions (`transition: all .3s ease`)
- Keyframe animation (`float`)
- Gradient text effects using `background-clip`
- Mobile responsive breakpoints (768px)
